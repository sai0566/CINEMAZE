## 📌 Project Overview

CINEMAZE is a frontend movie web application that allows users to browse movies, view detailed information, watch trailers, and manage their favorites list.

This project demonstrates strong understanding of:
- DOM manipulation
- Event handling
- LocalStorage
- Modular CSS structure
- Responsive design

---

## ✨ Features

### 🔐 Authentication
- User Registration
- Login validation
- Session stored in LocalStorage

### 🎬 Movie Browsing
- Display movie cards
- Responsive grid layout
- Smooth hover animations

### 🔍 Search Functionality
- Real-time movie filtering

### ⭐ Favorites System
- Add movies to favorites
- Prevent duplicate entries
- Data stored in LocalStorage

### 🎥 Movie Details Page
- Poster display
- Description
- Rating
- Trailer (YouTube iframe)
- Add to Favorites button

### 📱 Responsive Design
- Desktop
- Tablet
- Mobile friendly
